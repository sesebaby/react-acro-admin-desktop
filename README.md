## 简介

React Acro Admin 桌面端。

2024 年 12 月开源，旨在帮助企业、在校生及个人开发者快速上手开发 React 桌面端后台管理项目，提高开发效率。

## 技术栈

- 🎉 使用 `Vite` 构建
- 🎉 使用 `Typescript` 编写
- 🎉 使用 `Electron` 框架
- 🎉 使用 `React` 框架
- 🎉 使用 `Tailwindcss` Css 框架
- 🎉 集成 `React` 路由，`扁平化路由`设计
- 🎉 集成 `Zustand` 状态管理，支持状态持久化
- 🎉 集成 `AcroDesign` 企业级组件库，智能设计体系，连接轻盈体验

## 特性

- 🎉 支持 i18n 语言国际化
- 🎉 支持暗黑模式
- 🎉 支持主题定制
- 🎉 支持全屏切换
- 🎉 路由持久化管理
